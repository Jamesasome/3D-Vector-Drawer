# 3D-Vector-Drawer
Draws 3D vectors in PyOpenGL and PyQt6, also calculations the sum, difference and cross product of two 3D vectors and plots the result

### Features

- Using PyOpenGl, PyQt6, and pyqtgraph, plot 3D vectors, compute simple vector operations and plot the result

### Dependencies
- PyOpenGl
- PyQt6
- Numpy
